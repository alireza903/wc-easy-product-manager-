# wc-easy-product-manager-

**افزونه وردپرس + ووکامرس برای افزودن سریع محصول توسط کارمندان از طریق گوشی و ربات بله**

این افزونه امکان ثبت محصول را بسیار ساده کرده و مخصوص تیم فروش و کارمندان طراحی شده است.

---

## ✨ ویژگی‌ها

- **رابط کاربری ساده و موبایل‌فرندلی** (بهینه برای Mini App بله)
- افزودن محصول با عکس، قیمت، تخفیف، دسته‌بندی، موجودی و توضیحات
- **پشتیبانی کامل از ربات بله** (افزودن محصول از داخل چت ربات)
- ثبت Webhook خودکار ربات بله
- نمایش محصولات اخیر
- ایجاد نقش کاربری `product_staff` برای کارمندان
- آپلود عکس مستقیم از دوربین گوشی
- کاملاً ریسپانسیو و راست‌چین

---

## 📋 راهنمای استفاده برای کارمندان

### روش ۱: از طریق پنل وب (توصیه‌شده)

1. وارد سایت شوید و به صفحه **[پنل افزودن محصول](/product-panel/)** بروید.
2. ابتدا وارد حساب کاربری شوید (حساب باید دارای نقش `product_staff` باشد).
3. عکس محصول را بگیرید یا انتخاب کنید.
4. عنوان، قیمت، دسته‌بندی، توضیحات و موجودی را وارد کنید.
5. وضعیت انتشار را انتخاب کنید (منتشر شود یا پیش‌نویس).
6. روی **«افزودن محصول»** بزنید.

### روش ۲: از طریق ربات بله

1. ربات را باز کنید.
2. دستور `/add` را ارسال کنید.
3. مراحل را یکی یکی دنبال کنید (عنوان → قیمت → دسته‌بندی → توضیحات → عکس).
4. دستورات مفید:
   - `/add` → افزودن محصول جدید
   - `/list` → نمایش محصولات اخیر
   - `/cancel` → لغو عملیات
   - `/help` → نمایش راهنما

---

## 🛠️ نصب و راه‌اندازی

1. فایل افزونه را در پوشه `wp-content/plugins/` آپلود کنید.
2. در پیشخوان وردپرس افزونه **«WC Bale Product Manager»** را فعال کنید.
3. به منوی **«محصول آسان»** در پیشخوان بروید.
4. توکن ربات بله را وارد کنید.
5. Chat ID کارمندان را (هر کدام در یک خط) وارد کنید.
6. روی دکمه **«ثبت Webhook ربات بله»** کلیک کنید.

> **نکته:** بعد از فعال‌سازی، صفحه `/product-panel/` به صورت خودکار ایجاد می‌شود.

---

## ⚙️ تنظیمات ربات بله

- **دریافت توکن:** داخل بله به `@BaleBot` پیام دهید و دستور `/newbot` را بزنید.
- **دریافت Chat ID:** به `@BaleBot` دستور `/chatid` بدهید.
